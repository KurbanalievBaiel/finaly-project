# gσlobe - Travel Booking Website

Современный сайт бронирования путешествий, созданный с использованием React, Redux и Vite.

## Технологии

- **React 18** - UI библиотека
- **Redux Toolkit** - управление состоянием
- **Vite** - сборщик и dev-сервер
- **Font Awesome** - иконки

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Header/         # Навигационная панель
│   ├── Hero/           # Главная секция с фоном
│   ├── SearchWidget/   # Виджет поиска
│   └── Destinations/   # Секция с направлениями
├── store/              # Redux store
│   ├── slices/         # Redux слайсы
│   └── store.js        # Конфигурация store
├── App.jsx             # Главный компонент
├── main.jsx            # Точка входа
└── index.css           # Глобальные стили
```

## Функциональность

- Переключение между поиском рейсов и отелей
- Обмен местами отправления и назначения
- Управление состоянием через Redux
- Адаптивный дизайн
- Плавные анимации при наведении
